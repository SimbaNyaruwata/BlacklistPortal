
## About BlackListPortal

BlackListPortal is a web application that has been developed using Laravel and MySQL Database

## How to Setup BlackListPortal

- Make sure you have your  Xampp server up and running and have the following modules runnning APACHE and MYSQL. 
-  Import the database, that is in the project, in your xampp server side


## How to run the Project
- Run your project using "composer run dev" in the terminal
- Access the project using the url that the server is running on eg  [http://127.0.0.1:8000] displayed in the terminal
- First register a new user
